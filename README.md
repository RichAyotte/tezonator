# tezonator

To install dependencies:

```bash
bun install
```

## Usage
```bash
bun src/tezonator.ts [command] [network]
```

### Networks currently supported

- dailynet
- ghostnet
- mainnet
- oxfordnet
- parisnet
- predalnet
- weeklynet

### Build

```bash
bun src/tezonator.ts build mainnet
```

### Init

```bash
bun src/tezonator.ts init mainnet
```
