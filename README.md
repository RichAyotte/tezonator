# tezonator

To install dependencies:

```bash
bun install
```

## Usage
```bash
bun src/tezonator.ts [command] [network]
```

### Networks currently supported

- ghostnet
- mainnet
- oxfordnet
- parisanet
- parisbnet
- predalnet
- weeklynet

### Build

```bash
bun src/tezonator.ts build mainnet
```

### Init

```bash
bun src/tezonator.ts init mainnet
```

### Start

```bash
bun src/tezonator.ts start mainnet
```
